# Encrypt_Decrypt_file

Graphical-User-Interface based encryptor and file decryptor!

## Usage

1. Clone the project
````bash
git clone https://github.com/Ananya-Bhardwaj/Encrypt_Decrypt_file.git
````

2. Install Required Dependencies
````bash
pip install fernet
pip install tk
````

3. Run the program

````bash
python3 gui.py
````
